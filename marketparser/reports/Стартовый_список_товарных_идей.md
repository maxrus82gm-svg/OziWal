# Стартовый список идей брелков

> Навигация: [[marketparser/Карта_анализа_рынка|Карта анализа рынка]] · Проверка структуры: [[marketparser/reports/Проверка_чтения_товарных_идей|Проверка чтения товарных идей]]

Создан стартовый список идей для будущей проверки спроса на брелки, жетоны, подвески и небольшие аксессуары, которые можно моделировать и печатать на 3D-принтере.

Файл со списком:

`marketparser/data/input/product_ideas.csv`

## Включённые группы идей

- персональные брелки;
- животные и питомцы;
- знаки и символы;
- практичные брелки;
- безопасные игровые темы;
- рискованные игровые поисковые гипотезы.

## Игровые темы и IP-риск

Игровые темы разделены на две группы:

1. Безопасные общие игровые идеи: геймпад, WASD, пиксельный меч, кубик d20, игровой жетон и похожие оригинальные темы.
2. Рискованные брендовые поисковые гипотезы: Minecraft, Roblox, Dota, CS2, Genshin, Fortnite, Stalker, Warhammer, Skyrim, Fallout, Cyberpunk.

Рискованные игровые запросы нужны только для исследования спроса. Их нельзя считать безопасными производственными идеями.

Для производства нужно использовать безопасные альтернативы из колонки `safe_alternative`, например:

- `брелок minecraft` -> `пиксельный меч`;
- `брелок stalker` -> `постапокалиптический артефакт`;
- `брелок fallout` -> `ретро-футуристический жетон`;
- `брелок cs2` -> `брелок FPS / брелок для геймера`;
- `брелок genshin` -> `фэнтези-амулет стихии`;
- `брелок warhammer` -> `готический печатный жетон без символики бренда`.

## Важные ограничения

- Точный спрос ещё не измерялся.
- Количество товаров, цены, отзывы и рейтинги ещё не проверялись.
- Запросы к OZON и Wildberries не выполнялись.
- Парсер, код парсинга, API-клиенты и Excel-файлы не создавались.
- CSV является только стартовым списком идей для будущей ручной или полуавтоматической проверки.
