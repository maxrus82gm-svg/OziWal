# marketparser

`marketparser` — рабочая зона для будущего инструмента анализа спроса на товары OZON и Wildberries.

Главная цель:

- помогать выбирать, какие 3D-модели стоит моделировать и печатать;
- проверять товарные идеи до производства;
- собирать данные по спросу, конкуренции, ценам, отзывам и качеству карточек;
- формировать рейтинг перспективности товарной идеи.

На текущем этапе:

- парсер не реализован;
- код не создаётся;
- API не подключается;
- сайты не парсятся;
- создаётся только документация и правила структуры.

Вся работа по парсеру должна оставаться внутри папки:

`marketparser/`

Постоянные правила проекта находятся в:

`Support/`
