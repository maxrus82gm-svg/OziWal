# 04. Оценка товарных идей

> Навигация: [[marketparser/Карта_анализа_рынка|Карта анализа рынка]]

Этот документ описывает предварительную систему оценки товарной идеи.

## Что оценивается

Идею нужно оценивать по направлениям:

1. Спрос.
2. Конкуренция.
3. Сложность моделирования.
4. Сложность печати.
5. Время печати.
6. Расход материала.
7. Риск брака.
8. Лёгкость упаковки.
9. Лёгкость фото.
10. Возможность сделать лучше конкурентов.
11. Маржа.
12. Итоговый приоритет.

## Простая шкала

- `0` — плохо / нет данных;
- `1-3` — слабый показатель;
- `4-6` — средний показатель;
- `7-10` — сильный показатель.

## Пример формулы

```text
final_score = demand_score - competition_score + production_score + design_advantage_score + margin_score
```

Где:

- `demand_score` — сила спроса;
- `competition_score` — сила конкуренции;
- `production_score` — удобство моделирования, печати, упаковки и фото;
- `design_advantage_score` — шанс сделать товар лучше конкурентов;
- `margin_score` — примерная экономическая привлекательность.

## Важное уточнение

Формула предварительная. Она нужна для сравнения идей между собой, а не для точного прогноза продаж.

Итоговый приоритет должен помогать выбрать, какие 3D-модели стоит моделировать и печатать в первую очередь.
