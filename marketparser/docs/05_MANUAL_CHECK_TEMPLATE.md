## Как заполнять поля

- `idea` — идея товара.
- `query` — поисковый запрос.
- `marketplace` — OZON или WB.
- `price_min` — минимальная цена среди похожих товаров в топе.
- `price_max` — максимальная цена среди похожих товаров в топе.
- `reviews_max` — максимум отзывов у похожего товара в топе.
- `rating_avg` — примерный средний рейтинг.
- `competition_level` — низкая / средняя / высокая.
- `can_make_better` — да / нет / возможно.
- `print_difficulty` — низкая / средняя / высокая.
- `notes` — комментарий.

## Правило заполнения

Если данных не хватает, поле нужно оставить пустым или пометить как `ТРЕБУЕТ УТОЧНЕНИЯ`.

Не нужно выдумывать точные значения. Цель ручной проверки — быстро сравнить идеи между собой и выбрать 3-5 лучших вариантов для дальнейшей работы.
