# 00. Правила анализа рынка

> Навигация: [[marketparser/Карта_анализа_рынка|Карта анализа рынка]]

Этот документ фиксирует главные правила папки `marketparser`.

## Назначение папки

`marketparser` — отдельная рабочая зона для будущего инструмента анализа спроса на товары OZON и Wildberries.

Инструмент нужен для оценки товарных идей перед моделированием, печатью, фотографированием и созданием карточки товара.

## Главные правила

1. Все файлы парсера должны лежать только внутри `marketparser`.
2. Не создавать файлы парсера в корне репозитория.
3. Не создавать временные файлы вне `marketparser`.
4. Не создавать код без отдельной задачи.
5. Не подключать OZON / Wildberries без отдельной задачи.
6. Не обходить капчи, авторизацию, антибот-защиту или ограничения сайтов.
7. Не делать агрессивный парсинг.
8. Сначала используется ручная или полуавтоматическая проверка спроса.
9. Данные должны храниться структурированно.
10. Все результаты исследования должны быть воспроизводимыми: должно быть понятно, по какому запросу, на какой площадке и какие данные были собраны.

## Текущий этап

На текущем этапе поддерживаются документация, стартовый список товарных идей и результаты ограниченных ручных проверок.

Не создаются:

- рабочий парсер;
- Python-скрипты;
- JavaScript-скрипты;
- API-клиенты;
- Excel-файлы;
- подключения к маркетплейсам.
