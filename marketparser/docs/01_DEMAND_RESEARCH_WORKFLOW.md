# 01. DEMAND RESEARCH WORKFLOW

Этот документ описывает процесс проверки товарной идеи для OZON и Wildberries.

## Процесс проверки идеи

1. Пользователь или ассистент формирует список идей.
2. Для каждой идеи подбираются поисковые запросы.
3. Идеи проверяются на OZON / Wildberries.
4. Для каждого запроса собираются признаки спроса.
5. Для каждого запроса собираются признаки конкуренции.
6. Отдельно оценивается сложность моделирования и печати.
7. Идея получает итоговый приоритет.
8. Пользователь выбирает 3-5 лучших идей для моделирования и тестовой печати.

## Признаки спроса

Признаки спроса помогают понять, покупают ли похожие товары и есть ли у идеи рыночный интерес.

- много отзывов у товаров в топе;
- есть товары с рейтингом;
- есть повторяющиеся карточки;
- есть похожие поисковые подсказки;
- есть понятная покупательская потребность.

## Признаки конкуренции

Признаки конкуренции помогают понять, насколько сложно будет выделиться среди существующих товаров.

- очень много одинаковых товаров;
- низкие цены;
- сильные фото;
- много продавцов;
- много товаров с тысячами отзывов;
- сложно выделиться дизайном.

## Признаки хорошей идеи для 3D-печати

Хорошая идея для 3D-печати должна быть не только востребованной, но и удобной для производства небольшими партиями.

- небольшая модель;
- быстрый цикл печати;
- низкий риск брака;
- понятная польза;
- легко сфотографировать;
- можно сделать дизайн лучше конкурентов;
- есть возможность продавать набором;
- хорошая маржа.

## Результат проверки

Результатом проверки должен быть короткий список перспективных идей, которые можно сравнить между собой и выбрать для моделирования, тестовой печати и подготовки карточки товара.
