# 07. BIG TASK CONTEXT — большой контекст задачи

## Назначение документа

Этот файл используется для хранения большой задачи, длинного контекста или подробной инструкции.

Пользователь может вставлять сюда большую задачу от ассистента, а затем в чат Codex отправлять короткую маленькую задачу.

## Как использовать

1. Ассистент готовит большую задачу или подробную инструкцию.
2. Пользователь вставляет её в этот файл.
3. Потом пользователь даёт Codex короткую задачу в чат.
4. Если короткая задача требует учитывать большой контекст, Codex должен сначала прочитать этот файл.
5. Codex выполняет только маленькую текущую задачу, а не весь большой контекст целиком.

## Важное правило

Codex не должен автоматически выполнять всё, что написано в этом файле.

Этот файл является контекстом, а не текущей задачей.

Текущей задачей считается только то, что пользователь отдельно написал в чате Codex.

## Правило приоритета

Приоритет инструкций такой:

1. Текущая маленькая задача из чата Codex.
2. Правила из `Ozon/Support/00_START_HERE_CARD_RULES.md`.
3. Правила из `Ozon/Support/01_CODEX_LOCAL_RULES.md`.
4. Правила из `Ozon/Support/05_EXCEL_SAFETY_RULES.md`, если задача связана с Excel.
5. Большой контекст из `Ozon/Support/07_BIG_TASK_CONTEXT.md`.
6. Остальные документы проекта, если они указаны в маленькой задаче.
