# 05. EXCEL SAFETY RULES — правила безопасной работы с Excel

## Назначение документа

Этот документ описывает правила безопасности при работе с Excel-файлами в проекте `OziWal`.

Excel-файлы могут использоваться для подготовки, проверки или загрузки карточек товаров OZON.

## Главный принцип

Excel-файл нельзя менять без явной задачи.

Если задача не требует изменения Excel, Codex не должен открывать, пересохранять или модифицировать Excel-файлы.

## Что запрещено без отдельного разрешения

Codex не имеет права:

1. Менять структуру Excel-файла.
2. Удалять листы.
3. Переименовывать листы.
4. Менять порядок листов.
5. Менять порядок колонок.
6. Удалять колонки.
7. Удалять строки с данными.
8. Менять формулы.
9. Перезаписывать формулы значениями.
10. Менять типы данных в колонках.
11. Менять формат файла.
12. Пересохранять файл просто ради форматирования.
13. Массово заменять данные без отдельной задачи.
14. Удалять выпадающие списки или проверки данных.
15. Менять скрытые листы или служебные области без разрешения.

## Что разрешено только по задаче

Разрешено только если прямо указано в задаче:

- заполнить конкретные ячейки;
- добавить новые строки с товарами;
- обновить конкретные значения;
- экспортировать копию;
- создать новый Excel-файл по шаблону;
- проверить структуру без изменения файла.

## Обязательный порядок работы с Excel

Если задача требует изменения Excel, Codex должен:

1. Понять назначение файла.
2. Описать план изменения.
3. Проверить, какие листы и колонки затрагиваются.
4. При возможности сделать резервную копию.
5. Менять только указанные ячейки, строки или листы.
6. Не трогать формулы, если это не указано.
7. После работы сообщить, что именно изменено.

## Если есть сомнение

Если Codex не уверен, можно ли менять Excel-файл, он должен остановиться и задать вопрос.
