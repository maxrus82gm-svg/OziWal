# Рабочая среда проекта

## Назначение документа

Этот документ определяет текущую рабочую среду проекта для ChatGPT и других ИИ-агентов.

Документ позволяет агенту автоматически определить:

- текущее рабочее место;
- локальный путь к проекту;
- режим работы агента.

Документ является единственным источником этой информации для проекта.

# Режимы агента

## Самостоятельный

Агент самостоятельно:

- анализирует проект;
- принимает локальные технические решения;
- может работать сразу с несколькими файлами;
- выполняет комплексные изменения в рамках поставленной задачи;
- отвечает за качество итоговой реализации.

---

## Ограниченный

Агент:

- является исполнителем поставленной задачи;
- не принимает архитектурных решений самостоятельно;
- не изменяет дополнительные файлы без необходимости;
- не выполняет улучшения, отсутствующие в задании;
- при любой неоднозначности обязан остановиться и сообщить о проблеме.
