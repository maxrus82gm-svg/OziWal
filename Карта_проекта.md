# Карта проекта

Главная точка входа в документацию и рабочие материалы проекта `OziWal`.

Граф проекта строится по иерархии: карта проекта → тематическая карта → документ.

## Документация проекта

- [[Support/Карта_документации|Карта документации]] — управление проектом, агентский цикл, карточки товаров, Excel, шаблоны и примеры.

## Анализ рынка

- [[marketparser/Карта_анализа_рынка|Карта анализа рынка]] — правила, процессы, структура данных и результаты исследований `marketparser`.

Обычные документы открываются через соответствующую тематическую карту.
